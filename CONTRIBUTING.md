# Contributing to Stellar Utils

Thank you for your interest in contributing to Stellar Utils! Here's how you can help:

## Getting Started

1. Fork the repository
2. Clone your forked repo
3. Install dependencies: `npm install`
4. Make your changes
5. Write tests for your changes
6. Run tests: `npm test`
7. Submit a pull request

## Code Style

- Follow the existing code style
- Use meaningful variable and function names
- Add JSDoc comments for all functions
- Keep functions focused and small

## Pull Requests

1. Make sure your PR description clearly describes the changes
2. Link to any relevant issues
3. Ensure all tests pass
4. Wait for review and address any feedback

## Reporting Bugs

When reporting bugs, please include:
- Steps to reproduce
- Expected behavior
- Actual behavior
- Node.js version
- Any error messages

## Feature Requests

We're always open to new ideas! Feel free to open an issue to discuss potential features.
